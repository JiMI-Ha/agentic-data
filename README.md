# Agentic 数据合成阅读笔记

这是一个整理与沉淀 **agentic data synthesis** 相关论文阅读的仓库。内容以中文笔记为主，涵盖方法概述、关键设计、实验结论与个人理解，便于持续回顾与扩展。

## 目标

- 系统化记录 agentic 数据合成领域的重要论文与进展
- 提炼可迁移的设计模式与评估思路
- 形成可复用的阅读模板与长期知识库

## 仓库结构（计划）

- `papers/` 论文阅读笔记（按年份或主题归档）
- `summaries/` 主题综述与对比整理
- `resources/` 相关资料与链接（博客、项目、数据集等）
- `templates/` 阅读模板与记录规范

> 说明：结构会随阅读推进逐步完善。

## 阅读规范（建议）

每篇论文尽量包含以下要点：

- 研究问题与动机
- 方法核心与系统结构
- 数据合成流程与关键设计
- 实验设置与主要结果
- 局限与可改进点
- 个人思考与与其他工作的联系
